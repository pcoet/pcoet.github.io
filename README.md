# pcoet.github.io

## Install

    python3 -m venv mkdocs-env
    source mkdocs-env/bin/activate
    pip install mkdocs-material

## Activate

    source mkdocs-env/bin/activate

## Deactivate

    deactivate

## Build

    mkdocs build

## Serve

    mkdocs serve
